Safe Search Replace

A serialize-friendly search and replace script for MySQL databases, particularly useful for WordPress migrations

This project is a community fork of the "Search Replace DB v2.0.0" script available from interconnect/it. At the moment it contains minor bug fixes on the original script, but will deviate more over time.
